Normalmente as empresas possuem um ambiente de produção que é o ambiente da branch master, esse ambiente de produção é onde o produto ja está em uso pelo cliente.E um ambiente de desenvolvimento que é uma outra branch master criada onde os devs estão trabalhando. quando eu tenho uma tarefa devo criar uma branch dessa tarefa, e assim que ela for finalizada eu devo enviar essa tarefa para a branch de desenvolvimento, e depois de todos os teste ela sera enviada para a branch master (produção) que é o utilizada pelo cliente final
